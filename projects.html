OdinBook

Code Live

This is a knockoff of Facebook. I made this fairly basic, but with a lot of functionality.

Built using Ruby on Rails, deployed on heroku. When you sign up, you have the option to create a new account or login via Facebook.

I generated a mailer to welcome you when you create a new account.

I used Omniauth to allow you to login via Facebook.

I used AWS S3 via Active Storage to allow an attachment/update of an avatar photo.

I styled the site using Bootstrap.

I created a relational database to store users, posts, comments, friend requests, friendships, and likes.

You can use this by logging in with:

Email: 'test@test.com' Password: 'password'

Or feel free to create an account/login with Facebook!

To Do

Code Live

A ToDo list.

In this project, I created a TO DO list using HTML/CSS/JavaScript and webpack. I wrote the code in modules so that I could write clean, maintainable code.

All tasks/projects are saved locally using LocalStorage.

Projects are listed in tabs on the top, which have an event listener for deletion as well as selection.

Each task is color-coded based on priority.

Tasks can be created, edited, and deleted.

Library

Code Live

A virtual library.

In this project, I used JavaScript and HTML/CSS to create an app that allows you to add books to your virtual library. The library is saved locally via LocalStorage.

I used event listeners to give you the ability to change the 'read' status, as well as delete the book once already in the library. There is a button to add new books to the library as well as a modul that pops up to fill in the fields.

I also used event listeners to hide/show class styles as in the form for it to pop up and an image that explains to press the 'add' button, but the image only shows when there are no books in the library.
