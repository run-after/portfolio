OdinBook

Code Live

This is a knockoff of Facebook with a lot of functionality.

Built using MVC framework with Ruby on Rails, styled with Bootstrap, deployed on Heroku. When you sign up, you have the option to create a new account or login via Facebook.

I generated a mailer to welcome user when they create a new account.

I used Omniauth to allow login via Facebook.

I used AWS S3 via Active Storage to allow an attachment/update of an avatar photo.

I created a relational database to store users, posts, comments, friend requests, friendships, and likes.

You can use this by logging in with:

Email: 'test@test.com' Password: 'password'

Or feel free to create an account/login with Facebook!

Battleship

Code Live

The classic game of Battleship.

In this project I utilized TDD with Jest.

Packaged everything with Webpack.

Created custom CSS with animations and responsive design.

Using factory functions, modular design and a lot of event listeners, I created a functional game that works well.

I allowed the player to place ships with their mouse. They can click the orientation button to flip between 'x' and 'y', and click to place the ship. I included logic to not allow ships to overflow off the board, and also not overlap another ship.

When placing ships, the squares are highlighted to show where they will be placed, and once placed, the ship renders on the board so the player knows where other ships are located.

Memory Game

Code Live

This is a game to test your memory.

Built using React hooks, deployed on Github pages.

Play by clicking each card only once. If you select the same card twice, your score will go to 0 and you will get an alert stating that you selected a card twice.

I use event listeners to listen for a click on each card, modify state to ackowledge that this card has been chosen, then each click checks whether that card has been selected yet or not.

Chess

Code Live

A command line chess game.

You can either play this game locally if you have ruby installed, or play it on my repl.it.

If you play on repl.it, in the command line on the right, type "cd lib", then hit "enter". Then type "ruby game.rb".

If you play on locally, in the command line, navigate to chess/lib then type "cd lib", then hit "enter". Then type "ruby game.rb".

The game should allow you to play through. If you're playing locally, you can save the game at any time. I did this usling yaml.
